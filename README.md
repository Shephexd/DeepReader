# DeepReader - 개역한글 성경의 새로운 깊이

![SwiftUI](https://img.shields.io/badge/SwiftUI-Framework-blue.svg)
![SwiftData](https://img.shields.io/badge/SwiftData-Persistence-brightgreen.svg)
![Firebase](https://img.shields.io/badge/Firebase-Analytics-yellow.svg)

**DeepReader**는 하나님의 말씀을 가장 순수하고 깊이 있게 묵상할 수 있도록 설계된 프리미엄 성경 리더입니다. 절제된 디자인과 사용자 편의 기능을 통해 매일의 영적 여정을 소중히 기록해 보세요.

## ✨ 주요 기능

### 📖 전통의 가치를 담은 개역한글 & KJV
- 가장 경건한 문체를 유지하는 **개역한글** 버전 수록 (31,101구절 완비).
- 권위 있는 영문 성경인 **KJV(King James Version)** 나란히 대조 읽기.
- 단어 하나하나에 담긴 미세한 뉘앙스를 발견할 수 있는 입체적 묵상 환경.

### 📅 나만의 성경 읽기 여정, 달력 뷰
- 오늘 읽은 말씀이 자동으로 기록되는 시각적인 **Reading Calendar**.
- 매일 쌓이는 읽기 기록을 통해 나의 영적 성장을 한눈에 확인.

### 🖍️ 감각적인 형광펜과 저장 기능
- 기억하고 싶은 구절을 여러 색상의 **형광펜**으로 강조 (Yellow, Green, Peach).
- 소중한 말씀을 나만의 **보관함(Saved)**에 저장하여 언제든 다시 묵상.

### 🎨 프리미엄 UI/UX 및 다크 모드
- **타이포그래피:** 가독성이 뛰어난 세리프(Serif) 글꼴 및 영문 Georgia 폰트 적용.
- **다크 모드:** 밤낮 관계없이 눈이 편안한 환경에서 말씀에 집중.
- **iPad 최적화:** 넓은 화면에 맞춘 레이아웃과 내비게이션 구조 지원.

### 🚀 현대적 기술 스택
- **SwiftUI:** 최신 선언적 UI 프레임워크를 통한 부드러운 애니메이션.
- **SwiftData:** 고성능 온디바이스 데이터 영속성 관리.
- **Firebase Analytics:** 사용자 활동 분석을 통한 지속적인 품질 개선.

## 🛠 설치 및 시작하기

### 요구 사항
- iOS 17.0 이상
- Xcode 15.3 이상

### 설정 방법
1. 이 저장소를 클론합니다.
   ```bash
   git clone https://github.com/shephexd/DeepReader.git
   ```
2. `DeepReader.xcodeproj` 파일을 Xcode에서 엽니다.
3. **Firebase 설정:** 본인의 `GoogleService-Info.plist` 파일을 `DeepReader/` 폴더에 추가합니다.
4. **패키지 로드:** `File > Packages > Resolve Package Versions`를 실행하여 의존성 라이브러리를 다운로드합니다.
5. 시뮬레이터 또는 기기에서 실행합니다.
