# DeepReader 공식 마케팅 및 가이드 페이지

이 폴더는 **DeepReader (개역한글 성경)** 앱의 Apple App Store 심사 및 사용자 지원을 위한 공식 웹사이트 소스입니다. GitHub Pages를 통해 호스팅됩니다.

## 📁 주요 파일 설명
*   `index.html`: 앱의 주요 기능 소개 및 마케팅 페이지
*   `support.html`: 사용자 기술 지원 및 문의처 (App Store 필수 항목)
*   `privacy.html`: 개인정보 처리방침 (App Store 필수 항목 - Firebase 수집 내용 포함)

## 🚀 배포 방법 (GitHub Pages)
1.  이 폴더를 포함하여 GitHub에 `push` 합니다.
2.  GitHub 저장소의 **Settings > Pages** 메뉴로 이동합니다.
3.  **Build and deployment** 섹션에서:
    *   Source: `Deploy from a branch` 선택
    *   Branch: `main` (또는 현재 브랜치) 및 폴더를 `/(root)` 또는 `/docs`로 설정합니다.
    *   *참고: 만약 별도 저장소가 아닌 현재 프로젝트의 하위 폴더로 사용한다면, 마케팅 페이지 주소는 `https://<유저명>.github.io/DeepReader/lsguide/`가 됩니다.*

## 📝 심사 제출용 URL 정보
*   **Marketing URL:** `.../lsguide/index.html`
*   **Support URL:** `.../lsguide/support.html`
*   **Privacy Policy URL:** `.../lsguide/privacy.html`
